## 2026-07-07 — Stabilization: the Bluetooth TCC kill and the frozen main thread, version 0.8.1

The founder halted feature work after 0.8.0 crashed repeatedly under real use: the app died within a minute of every launch, and the Grant Permission button beachballed before the exit. Iterations 10 to 12 stay designed and queued; this entry is purely fixes.

Progress: the five crash reports all carried one signature, a TCC privacy kill. The new BluetoothCollector enumerated paired devices through IOBluetooth every 30 seconds, that call is TCC-protected, and the packaged Info.plist had no NSBluetoothAlwaysUsageDescription, which macOS punishes with SIGABRT rather than a prompt. The fix is two-layered: the usage description now ships in the Info.plist, and the collector fail-closes by checking the read-only, non-prompting CBCentralManager authorization state before every tick, so an unauthorized process can never reach IOBluetooth in any context (the swift-run and test harnesses have no Info.plist at all). Unauthorized ticks book an honest needs-permission state, and the app still never raises the Bluetooth prompt itself, so the family stays dormant until an explicit enable affordance ships. The slow Grant Permission was a separate bug: the permission request is a blocking XPC round trip and the reset flow spawns tccutil synchronously, and both ran on the main actor; they now run detached with only the published outcome hopping back. Launch also stopped blocking the main thread: both collector registries start on a background queue, after an audit confirmed no collector needs a main run loop. A regression test pins the fail-closed gate, and a repo-wide sweep found no other TCC-protected API and no other main-thread blocking work. Ground truth: 386 tests green, a ten-minute soak of the packaged app with no crash report (0.8.0 died within sixty seconds), and one adversarial review of the diff. The packaged 0.8.1 runs locally; the tap publish deliberately waits for the founder's verdict tomorrow (a single scripts/release.sh run), and iteration 10 (honest numbers) resumes on his word.
